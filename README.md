# Stick Master Game 
<img width="1512" height="858" alt="Screenshot 2026-04-17 at 7 22 50 PM" src="https://github.com/user-attachments/assets/feefa000-38b7-41c3-9a25-8efef1d01d34" />

A fun and addictive web-based game where you help the hero cross platforms by stretching a stick to the perfect length. This version features custom background themes and sound effects!

#Features ✨
Dynamic Backgrounds: The game randomly switches between three beautiful themes (Day, Night, and Sunset) every time you restart.
Immersive Sound: Includes a custom "sad beep" sound effect when the character falls.
Smooth Gameplay: Built with HTML5 Canvas for fluid animations.
High Score Tracking: Challenge yourself to beat your previous best!

#How to Play 🕹️ 
1.  Start:** Click or tap on the screen to begin.
2.  Stretch: Hold down the mouse button (or spacebar) to grow the stick.
3.  Release: Release the button to let the stick fall. 
4.  Goal: Ensure the stick is long enough to reach the next platform, but not too long to overshoot it.
5.  Restart: If you fall, click the "RESTART" button to try again with a new random background.

#Technologies Used 🛠️ 
HTML5: For the game structure and Canvas rendering.
CSS3: For styling the UI elements and layout.
JavaScript (ES6): For the game logic, physics, and Web Audio API integration.

#Installation & Setup
No installation is required! You can run this game locally on your computer:
Clone this repository or download the ZIP file.
Open the `index.html` file in any modern web browser (Chrome, Firefox, Edge, etc.).
